git init [to initialise git in a new folder]
git status [ to check the untracked changes]
git add -A [to add changes to staging area]
git commit -m 'message'[to commit changes to local repository]
git log [to get the history of changes made]
git stash [ to recover the code]
git push [adding local repository content to remote repository]
git clone [exact copy from remote repository to local repository]
git pull [changes made in remote repository added to local repository]
git remote add remotename path[ to add remote repository to local repository]
git push remotename master [to push local repository contentb to remote repository]
git remote -v [to check the remote is added or not]
git pull remotename master [to pull the changes from remote repository to local repository]
